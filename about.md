Project Abstract: Umbraco Auto-Pilot
Achieved So Far:
We have successfully bridged the gap between Frontend Design and Backend CMS implementation. We built a Next.js Automation Dashboard that acts as a "Translator."

Intelligence Layer: Integrated Gemini 2.0 and OpenAI to analyze raw HTML and automatically map it to Umbraco-specific Data Types (Textstring, MediaPicker, etc.).

Infrastructure Layer: Developed a custom Umbraco API (C#) that receives JSON schemas and programmatically creates Document Types in the database while simultaneously generating the .cshtml Razor files on the disk.

Developer Experience: Eliminated the manual "Click-Ops" of Umbraco setup, reducing component creation time from 30 minutes to 30 seconds.
